# 🎬 Script de Vidéo de Démonstration : SmartAir Cameroon
**Livrable 3 - Hackathon IndabaX Cameroon 2026**
*Durée maximale : 3 minutes.*

---

## 🕒 Minute 1 : Problématique & Solution
- **Action** : Commencez par montrer la page "Accueil" du Dashboard avec le slogan.
- **Script** : "Au Cameroun, la qualité de l'air est un défi invisible. Entre l'Harmattan et les pics de chaleur, nos poumons sont en première ligne. Bonjour, nous sommes l'équipe SmartAir, et nous avons créé une IA pour la résilience climatique."
- **Action** : Montrer brièvement le Rapport Technique (XGBoost).
- **Script** : "Grâce à un modèle XGBoost entraîné sur 5 ans de données, nous avons atteint une précision de 99.8% pour prédire la pollution PM2.5 là où il n'y a pas de capteur physique."

## 🕒 Minute 2 : Démonstration du Dashboard & Carte
- **Action** : Naviguer vers l'onglet "Carte". Montrer les points pulsés.
- **Script** : "Regardez notre carte interactive. Ici, nous surveillons le réseau national. Les zones d'influence nous montrent où le risque est le plus élevé en temps réel."
- **Action** : Cliquer sur un marqueur (ex: Garoua ou Douala). Montrer le popup.
- **Script** : "Chaque ville a son bulletin de santé. Nous intégrons les directives de l'OMS pour déclencher des alertes automatiques."
- **Action** : Naviguer vers le "Dashboard" principal. Faire défiler les graphiques de corrélation.
- **Script** : "Le tableau de bord permet de corréler le climat et la pollution. Voyez ici comment la hausse des températures stagne les particules fines, guidant ainsi les décisions de santé publique."

## 🕒 Minute 3 : Simulateur & Impact
- **Action** : Aller sur la page "Predictions". Sélectionner une ville, cliquer sur l'icône de rafraîchissement ("Charger conditions actuelles"). Lancer la prédiction.
- **Script** : "Notre simulateur permet à n'importe quel agent de santé de tester des scénarios ou de charger les conditions actuelles du terrain en un clic pour obtenir une prévision immédiate."
- **Action** : Terminer sur la page "Impact".
- **Script** : "SmartAir n'est pas seulement un outil de prévision, c'est un bouclier pour le Cameroun. Des alertes précoces pour les hôpitaux à l'aide à la plantation d'arbres, nous mettons l'IA au service de la vie. Merci pour votre attention, et respirez l'avenir avec SmartAir Cameroon."

---

### 🎥 Conseils pour l'enregistrement :
1.  **Audio** : Utilisez un micro de bonne qualité (celui du kit mains libres suffit souvent).
2.  **Logiciel** : Utilisez OBS Studio ou Loom pour enregistrer votre écran.
3.  **Rythme** : Parlez calmement, la manipulation du dashboard doit être fluide.
